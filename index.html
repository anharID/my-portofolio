Portofolio

Project Terbaru

Berikut merupakan beberapa aplikasi yang telah saya pelajari dan saya kerjakan

Landing Page V.1

Ini merupakan halaman web landing page untuk portofolio yang pertama saya kerjakan. Ini dibuat menggunakan HTML dan kerangka kerja Bootstrap 5.

Movie App

Movie App merupakan aplikasi web yang dapat mencari movie atau film yang sedang tranding, yang akan datang , serta film top rated. Aplikasi web ini dibuat menggunakan framework React JS.

Blog App

Blog App merupakan aplikasi web blog sistem, dimana pengguna dapat menulis postingan atau blog mereka disini. Aplikasi blog sistem ini dibuat dengan Kerangka Kerja Laravel.

Todo List App

Todo List App merupakan aplikasi web todo untuk membuat daftar tugas yang sangat sederhana. Aplikasi ini dibuat menggunakan HTML, CSS, dan JavaScript.

Asset Management App

Asset Management App ini merupakan aplikasi web yang dapat mengelola aset perusahaan. Aplikasi ini saya buat menggunakan kerangka kerja Laravel untuk keperluan Kerja Praktik yang saya laksanakan di Perhutani Semarang.

Simple Calculator

Merupakan kalkulator sederhana berbasis web yang dengan riwayat perhitungan. Dibuat menggunakan HTML, CSS, dan JavaScript

Bookshelf App

BookShelf App merupakan aplikasi yang dapat mengelola buku mana yang telah kita baca atau yang belum kita baca. Aplikasi ini saya buat menggunakan HTML, CSS, dan JavaScript untuk keperluan submission pada platform pembelajaran Dicoding.
