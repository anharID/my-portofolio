Sistem Presensi Menggunakan Code

Sistem presensi menggunakan code di LAB IT merupakan sebuah presensi dengan input code yang telah dibuat untuk dapat menyatakan kehadiran. Terdapat 4 role user (Admin, Staf, PJ, dan Asisten). Pada sistem, user dapat generate code untuk dapat digunakan oleh user lain untuk dapat menyatakan kehadiran, melihat kode yang telah digunakan atau belum, 1 kode hanya dapat digunakan oleh 1 user dan hanya dapat digunakan 1 kali. Admin memiliki akses CRUD untuk data kelas, materi, dan asisten lab. Report presensi dapat di export ke spreadsheet untuk dapat diolah lebih lanjut.

Tech stack: HTML, CSS, JavaScript, Bootstrap, PHP, Laravel 9, MySQL

Sistem ERP Tambak Udang Vannamei STP UNDIP

Sistem ERP (Enterprise Resource Planning) Tambak Udang Vannamei STP UNDIP merupakan sebuah sistem informasi yang dikembangkan untuk membantu pengelola tambak udang dalam melakukan manajemen proses bisnis dalam satu sistem. Sistem ERP ini memiliki berbagai fitur diantaranya manajemen akun pengguna, manajemen karyawan, manajemen keuangan, manajemen inventaris dan logistik, manajemen peralatan, dan manajemen tambak udang. Proyek ini dikembangkan dalam tim 3 orang dengan jobdesk berdasarkan fitur. Saya bertanggung jawab dalam pengerjaan fungsi login dan autentikasi, manajemen akun, dan fitur pada manajemen tambak udang.

Tech stack: HTML, CSS, Tailwind CSS, JavaScript, PHP, Laravel 9, MySQL

Github Demo

Asset Management Information System

Sistem Informasi Manajemen Aset merupakan proyek yang dikerjakan saat menjalani kerja praktik di Perum Perhutani Divisi Regional Jawa Tengah yang berlokasi di Kota Semarang. Tujuan dikembangkannya sistem informasi manajemen aset ini yaitu untuk memudahkan dalam mengelola data yang berhubungan dengan asset yang dimiliki oleh Perhutani Divisi Regional Jawa Tengah. Dalam proses pengerjaan proyek dilakukan bersama tim 2 orang. Saya bertanggung jawab dalam merancang basis data dan back-end sistem dan memastikan sistem dapat berfungsi dengan baik sesuai dengan harapan.

Tech stack: HTML, CSS, Bootsrap, JavaScript, PHP, Laravel 8, MySQL

Github

Movie App

Movie App merupakan proyek tugas akhir pada praktikum rekayasa perangkat lunak berbasis komponen yang dikerjakan dalam tim 4 orang. Movie App adalah sebuah aplikasi React JS yang dapat mencari film. Pada aplikasi ini akan ditampilkan film rekomendasi, film yang akan datang, dan film dengan rating terbaik. User dapat mencari film dengan memasukkan kata kunci di kotak pencarian. Untuk data yang ditampilkan pada Movie App ini diambil dari TMDB API. Dalam proyek ini saya bertanggung jawab untuk membuat beebrapa komponen dan menampilkan data dari API serta routing pada aplikasi.

Tech stack: HTML, CSS, JavaScript, React JS, Material UI

Github Demo

Xiomay Store
